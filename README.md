## 프로젝트 설명
AI 연인이 삐진 이유를 15턴 안에 찾아내는 채팅식 게임 웹앱 서비스입니다.

원하는 상황을 선택 후 채팅 -> 분석하기로 대화와 AI연인의 감정을 분석할 수 있습니다.

아이디어/디자인 참고: [명절 잔소리 마스터](https://holiday-jansori.vercel.app/)
